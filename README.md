# simulator

※まだ動作確認していません...

##ISA

RV32I

##ビルド方法

`make`

##実行方法

シミュレートしたいファイルを`instr_file`としたときに
`./sim instr_file`

命令ファイルは必須です。ただし、バイナリでなくてもオプションで設定すれば文字でかかれた機械語でも平気です。

命令を一つずつ読んで実行していき、命令語がすべて0からなるものを読んだら停止し、すべてのレジスタの中身を整数で表示することになっています。（実行は一気に行われます）

##option

-t txt-instruction-file
-d data-file
-e txt-data-file
